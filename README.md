# DataStructures


## Description

Data structures code for the postgraduate entrance exams.


## Content

* sequence list
* single linked-list
* double linked-list
* sequence stack
* linked stack
* round-robin queue
* linked queue
* union-find set
* Huffman tree
* string matching algorithm (KMP, violence matching, optimized violence matching)
* sorting algorithm (shell, bubble, merge, selection, quick)
* AVL tree
* Heap
* B-Tree
